# Asset Property Editor

---

## Introduction

Asset Property Editor (APE) is a tool provided within the BO3 Mod Tools that allows modders to create, modify, or view the properties of game assets. It is commonly used when importing external assets to be used in [Radiant](/docs/black_ops_iii/radiant), characters or scripts. All asset properties are saved in GDT files. 

The UI can be customized so that widgets can be closed or rearranged. To restore closed widgets, navigate to `View -> Toolbars`.

### Asset Preview
The preview allows you to preview the asset that you currently have selected. This goes for models, textures, animations and FX.

## Useful information
 - Assets must have unique names, non-dependant of the GDT file.
 - NEVER press `save all` when closing the application. This can edit properties of core assets that should not be modified and will potentially corrupt your Mod Tools install.
 - If you want to make a change to assets in bulk, select all the assets, change the property and Ctrl + Click the blue arrow to the right of the property to apply it to all selected assets.
