# Getting Started with UI Development

---

Creating custom UI in Black Ops 3 can seem intimidating at first, but once you understand the basic setup, it becomes much easier to build your own HUDs, menus, text displays, and images. This guide assumes a basic level of scripting knowledge or higher.

---

The next sections will walk you through the fundamentals of setting up Lua-based UI files within your map. you will learn how to:

- Load custom Lua files into your map.
- Create and implement new UI widgets.
- Display basic UI elements on screen such as text and images.

---

After that, we'll dive deeper into connecting UI elements to live game data, such as:

- Ammo counters
- Player score
- Custom data sent from your own scripts

---

### Requirements

In order to work with Lua UI in Black Ops 3, you will need to download the following files:

1. [L3akMod](https://dtzxporter.com/tools/l3akmod)

Visit the L3akMod download page and follow the installation instructions provided there.
