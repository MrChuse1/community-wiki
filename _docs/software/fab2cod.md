# Fab2Cod


Fab2Cod is a converter tool that allows you to drag and drop USDZ models downloaded from [Fab](https://www.fab.com) and [SketchFab](https://sketchfab.com/). It will recreate the model, materials, and textures into your Black Ops III root directory under `./_custom/Fab`.

## Download

The latest version of Fab2Cod can be found on GitHub:
- Download [Fab2Cod](https://github.com/MrChuse1/Fab2Cod/releases/tag/latest)
